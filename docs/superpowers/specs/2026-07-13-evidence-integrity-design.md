# 墨西哥新品雷达证据完整性设计

## 目标

在没有 FastMoss、Nubimetrics 等付费接口权限的前提下，把每日 TikTok Shop Mexico、Temu Mexico、Mercado Libre Mexico 选品简报改造成可审计、可降级、不会把推测伪装成事实的公开页面。

## 数据策略

- 优先使用公开且可访问的官方平台、官方趋势工具和品牌/制造商资料。
- “热卖”或“爬升”至少需要两条独立需求证据，其中至少一条明确面向墨西哥。
- 销量、GMV、评分和评论数只有在官方或已授权数据源可核验时才展示具体数字；否则展示“暂无可核验数据”。
- 来源分为 A（官方平台或官方 API）、B（获授权第三方）、C（独立趋势代理）、D（供应商或技术资料）。
- 每条来源记录发布方、覆盖指标、地区、时间窗口、核查时间、是否估算和访问状态。
- 404、验证码墙、仅登录可见、搜索页和标签页不能作为可访问的直接证据。

## 自动替补

- TikTok：有 FastMoss Token 时启用 FastMoss；无 Token 时使用 TikTok Shop/Creative Center 可访问页面、具体公开视频和 Google Trends Mexico，且不声称准确 GMV。
- Temu：使用可访问的 Temu 商品快照、Google Trends Mexico、墨西哥跨平台同类需求及制造商规格；没有授权来源时不声称 Temu 销量。
- Mercado Libre：有官方开发者 Token 时启用 Trends API；无 Token 时使用可访问的公开页面与 Google Trends Mexico。验证码墙会标记为受限并由替补来源取代。
- FastMoss、Mercado Libre API、Nubimetrics 均预留适配器配置，但默认关闭，不绕过登录、订阅、验证码或限流。

## 页面表现

- 每个产品显示证据覆盖率、可信度、有效来源数、墨西哥本地证据数和最后核查时间。
- 每条来源显示等级、访问状态、来源类型、覆盖指标和“官方/第三方估算/推断”标签。
- 图片使用可访问缩略图并保留原始来源；视频必须链接到具体视频页。没有可验证视频时明确显示暂无素材。
- 失效来源保留审计记录，显示“已失效/登录受限/已替代”，不再作为评分依据。
- 分数解释公开；证据覆盖不足时总分封顶，不能成为 TOP 1。

## 发布与验证

- 数据校验阻止缺少来源元数据、独立证据不足或使用不可访问来源支撑精确指标的日报发布。
- 自动测试覆盖证据门槛、评分封顶、页面标签、媒体降级和静态站构建。
- 发布前运行完整测试、静态页面构建和链接结构检查。
